# Restaurant Management System

A simple Windows Forms application developed with C# and SQL Server.

## Features
- Customer management (Insert, Update, Soft Delete)
- Order registration
- Sales report per customer
- SQL Server Stored Procedures & Functions
- Parameterized queries

## Technologies
- C# (Windows Forms)
- SQL Server
- ADO.NET

## Database
Database scripts are available in the Database folder:
- CreateTables.sql
- StoredProcedures.sql
- Functions.sql
